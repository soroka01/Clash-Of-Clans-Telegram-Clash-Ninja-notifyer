# Clash Ninja Telegram Notifier

[English version](README_EN.md) · [GitHub](https://github.com/soroka01/Clash-Of-Clans-Telegram-Clash-Ninja-notifyer)

Telegram-бот на aiogram для отслеживания улучшений Clash of Clans через [Clash Ninja Upgrade Tracker](https://www.clash.ninja/upgrade-tracker). Он показывает текущие таймеры, обновляет одно сообщение без спама и присылает уведомления, когда улучшения завершены.

> [!IMPORTANT]
> Боту нужен `Cookie` авторизованной сессии Clash Ninja. Это секрет: не отправляйте его другим людям и не публикуйте `config.json`.

## Что умеет бот

- Отправляет уведомления о завершённых улучшениях построек, лаборатории и Pet House.
- Отмечает помощников прямо около улучшений: 🧑‍🔬 Lab Assistant, 👷 Builder's Apprentice и ⚗️ Alchemist.
- Показывает отдельный блок, если помощник действительно свободен. Окончание cooldown само по себе не считается освобождением и не вызывает уведомление.
- Открывает единый экран всех аккаунтов или отдельный экран каждого аккаунта.
- Обновляет открытый экран каждые 10 секунд редактированием сообщения, без новых сообщений в чате.
- Позволяет выбрать часовой пояс `UTC−12…UTC+14` прямо в меню; настройка хранится отдельно для каждого чата.
- Не редактирует старое меню после уведомления: следующее нажатие создаст новое актуальное меню.
- Хранит последний снимок Clash Ninja и настройки в SQLite, поэтому после перезапуска не теряет состояние.

## Требования

- Windows;
- Python 3.14 или новее;
- аккаунт Clash Ninja с заполненным Upgrade Tracker;
- Telegram-бот, созданный через [@BotFather](https://t.me/BotFather).

Git необязателен: `start.bat` умеет обновляться и без него.

## Быстрый запуск

1. Склонируйте репозиторий или скачайте [ZIP-архив](https://github.com/soroka01/Clash-Of-Clans-Telegram-Clash-Ninja-notifyer/archive/refs/heads/main.zip) и распакуйте его.
2. Создайте `config.json`, скопировав [config.example.json](config.example.json).
3. Заполните токен Telegram, свой Telegram ID и Cookie Clash Ninja.
4. Запустите [start.bat](start.bat) двойным кликом или из PowerShell:

   ```powershell
   .\start.bat
   ```

5. Откройте диалог с ботом в Telegram и отправьте `/start`.

При первом запуске скрипт создаёт `.venv`, обновляет `pip`, `setuptools` и `wheel` только внутри виртуального окружения, устанавливает зависимости и запускает бота. Глобальные Python-пакеты не используются.

## Настройка `config.json`

```json
{
  "bot_token": "PUT_TELEGRAM_BOT_TOKEN_HERE",
  "authorized_user_ids": [123456789],
  "notification_chat_ids": [123456789],
  "poll_interval_seconds": 60,
  "dashboard_refresh_seconds": 10,
  "utc_offset_hours": 5,
  "database_path": "data/clash_ninja_bot.sqlite3",
  "clash_ninja": {
    "tracker_url": "https://www.clash.ninja/upgrade-tracker",
    "cookie_header": "PUT_FULL_COOKIE_HEADER_FROM_LOGGED_IN_CLASH_NINJA_HERE",
    "request_timeout_seconds": 30
  }
}
```

| Поле | Назначение |
| --- | --- |
| `bot_token` | Токен от BotFather. |
| `authorized_user_ids` | Telegram ID пользователей, которым доступно меню. |
| `notification_chat_ids` | Личные чаты или группы для уведомлений. В группе сначала добавьте бота. |
| `poll_interval_seconds` | Как часто бот получает свежие данные Clash Ninja. По умолчанию — 60 секунд. |
| `dashboard_refresh_seconds` | Частота перерисовки уже открытого экрана. По умолчанию — 10 секунд. |
| `utc_offset_hours` | Часовой пояс по умолчанию: `5` — `UTC+5`, `3` — `UTC+3`, `-4` — `UTC-4`. Его можно изменить в боте для каждого чата. |
| `database_path` | Путь к локальной SQLite-базе. Не удаляйте её, если хотите сохранить историю состояния и настройки. |
| `clash_ninja.cookie_header` | Полное значение HTTP-заголовка `Cookie` от авторизованной сессии Clash Ninja. |

## Как получить Cookie Clash Ninja

Нужно именно полное значение HTTP-заголовка `Cookie`, а не файл cookie браузера.

1. Войдите в Clash Ninja и откройте [Upgrade Tracker](https://www.clash.ninja/upgrade-tracker).
2. Нажмите `F12` и перейдите на вкладку **Network** / «Сеть».
3. Обновите страницу: `Ctrl+R`.
4. Выберите запрос `upgrade-tracker` со статусом `200`.
5. Откройте **Headers** → **Request Headers**.
6. Скопируйте всё значение после `Cookie:` в одну строку.
7. Вставьте его в `clash_ninja.cookie_header` в `config.json` и сохраните файл.

Cookie имеет срок действия. Если в логе появляется сообщение, что Clash Ninja отклонил сессию, войдите на сайт заново и повторите эти шаги.

## Меню и уведомления

Отправьте `/start`, чтобы открыть главное меню.

- **Текущие улучшения** — экран всех аккаунтов; кнопками можно выбрать один аккаунт.
- **Настройки времени** — выбирает часовой пояс этого Telegram-чата.
- **Clash Ninja** — открывает Upgrade Tracker в браузере.
- **GitHub** — открывает репозиторий проекта.

На экране улучшений помощник отображается рядом с целью, например `🧑‍🔬 Freeze Spell`. Если помощник свободен, появляется блок «🤖 Свободные помощники», а бот также присылает отдельное уведомление. Помощник, который просто вышел из cooldown, не выдаётся за свободного от завершённого улучшения.

После перезапуска бот восстанавливает последнее сохранённое меню по его ID. Если его удалили или Telegram больше не разрешает его редактировать, бот автоматически отправляет замену с тем же экраном. Если меню ещё не было, главное меню отправляется в чаты из `notification_chat_ids`.

## Автообновление

Каждый запуск `start.bat` проверяет обновления.

- **Есть Git и это Git-репозиторий:** применяется только безопасное обновление `git pull --ff-only`. При локально изменённых исходниках обновление пропускается, чтобы не затереть их.
- **Git нет или проект распакован из ZIP:** скрипт скачивает актуальный ZIP с GitHub через встроенный PowerShell и обновляет исходники.

В обоих случаях `config.json`, `.venv`, `data/` и `logs/` сохраняются. В ZIP-режиме не редактируйте исходники вручную: при следующем обновлении они могут быть заменены актуальной версией из GitHub.

## Файлы, которые важно не терять

| Путь | Что хранится |
| --- | --- |
| `config.json` | Токен бота и Cookie Clash Ninja. Не попадает в Git. |
| `data/clash_ninja_bot.sqlite3` | Последний снимок, открытые экраны и настройки времени чатов. |
| `logs/bot.log` | Подробный журнал работы бота. |
| `logs/error.log` | Только ошибки для быстрой диагностики. |

## Решение проблем

| Симптом | Что сделать |
| --- | --- |
| `Clash Ninja отклонил сессию` | Обновите `cookie_header` по инструкции выше. |
| Бот не отвечает на `/start` | Проверьте `bot_token`, затем добавьте свой числовой Telegram ID в `authorized_user_ids`. |
| Нет уведомлений | Проверьте `notification_chat_ids`, что бот может писать в этот чат, и файл `logs/error.log`. |
| Не ставится обновление | Проверьте интернет. Без Git скрипт использует PowerShell и продолжит запускать текущую версию, если GitHub недоступен. |
| Таймеры или время не совпадают | Откройте «⚙️ Настройки времени» в боте и выберите нужный UTC. |
