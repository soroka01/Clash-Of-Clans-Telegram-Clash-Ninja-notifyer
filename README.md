# Clash Ninja Telegram Notifier

[English version](README_EN.md)

Telegram-бот на aiogram, который читает авторизованную страницу Clash Ninja
Upgrade Tracker, показывает активные улучшения и отправляет уведомления, когда
они завершаются.

## Возможности

- Уведомления о завершённых постройках, исследованиях Laboratory и улучшениях
  Pet House.
- Отдельные уведомления, когда после **реального улучшения** освобождаются Lab
  Assistant, Builder's Apprentice или Alchemist. Окончание их cooldown не
  считается завершением улучшения и не вызывает уведомление.
- Главное меню `/start` с переходом к общему списку, карточкам отдельных аккаунтов и настройкам времени.
- Окно со статусом редактируется каждые 10 секунд: обратный отсчёт обновляется
  без новых сообщений.
- Если после меню пришло уведомление, следующее нажатие создаёт новое меню, а
  не изменяет старое сообщение рядом с уведомлением.
- Снимок хранится в SQLite: после перезапуска бот продолжает сравнение с
  предыдущим состоянием.

## Требования

- Windows;
- Python 3.14+;
- аккаунт Clash Ninja с заполненным Upgrade Tracker;
- Telegram-бот, созданный через BotFather.

## Быстрый запуск

1. Скопируйте [config.example.json](config.example.json) в `config.json`.
2. Заполните токен, свои Telegram ID и Cookie Clash Ninja по инструкции ниже.
3. Дважды нажмите [start.bat](start.bat) или выполните:

   ```powershell
   .\start.bat
   ```

Скрипт сам создаст `.venv` в папке проекта, обновит `pip`, `setuptools` и
`wheel` **внутри него**, установит зависимости и запустит бота. Глобальные
пакеты для запуска проекта не используются. Перед запуском `start.bat`
безопасно проверяет GitHub и устанавливает доступные обновления через
`git pull --ff-only`. `config.json`, SQLite-база из `data/` и логи не
затрагиваются: они исключены из Git. Если вы изменили отслеживаемые файлы
проекта вручную, обновление пропускается, чтобы не потерять эти изменения.
Если Git не установлен (или проект был скачан ZIP-архивом), скрипт скачивает
актуальный код с GitHub через встроенный PowerShell. Локальные `config.json`,
`.venv`, `data/` и `logs/` при этом сохраняются.

## Настройка `config.json`

```json
{
  "bot_token": "токен от BotFather",
  "authorized_user_ids": [123456789],
  "notification_chat_ids": [123456789],
  "utc_offset_hours": 5,
  "clash_ninja": {
    "cookie_header": "полное значение заголовка Cookie"
  }
}
```

- `authorized_user_ids` — пользователи, которым доступно меню.
- `notification_chat_ids` — чаты или группы, в которые бот шлёт уведомления.
  Сначала добавьте туда бота и отправьте ему любое сообщение.
- `utc_offset_hours` — часовой пояс как смещение от UTC: например, `5` для
  `UTC+5` (Екатеринбург), `3` для `UTC+3`, `-4` для `UTC-4`. В нём выводятся
  время обновления, плановое время завершения и время в уведомлениях.
  После запуска это же значение можно изменить для каждого чата через
  «⚙️ Настройки времени» в главном меню; настройка сохраняется в SQLite.
- `cookie_header` — секрет сессии вашего аккаунта Clash Ninja. Не отправляйте
  его другим людям и не добавляйте `config.json` в Git.

## Где взять Cookie Clash Ninja

Для бота нужен не файл cookie-браузера, а **полное значение HTTP-заголовка
`Cookie`** у уже авторизованного запроса. Это надёжнее: базы cookie браузеров
зашифрованы ОС и не подходят для прямого копирования.

### Chrome, Edge, Opera, Яндекс Браузер

1. Войдите в Clash Ninja и откройте https://www.clash.ninja/upgrade-tracker.
2. Нажмите `F12` → вкладка **Network** / «Сеть».
3. Перезагрузите страницу (`Ctrl+R`).
4. В списке запросов выберите `upgrade-tracker` со статусом `200`.
5. Справа откройте **Headers** → **Request Headers**.
6. Скопируйте всё значение строки `Cookie:` — только текст после двоеточия.
7. Вставьте его одной строкой в `clash_ninja.cookie_header` в `config.json`.

Если всё же нужен путь к браузерной базе, в Chrome это обычно
`%LOCALAPPDATA%\Google\Chrome\User Data\Default\Network\Cookies`, в Edge —
`%LOCALAPPDATA%\Microsoft\Edge\User Data\Default\Network\Cookies`, а в
Firefox — `%APPDATA%\Mozilla\Firefox\Profiles\<профиль>\cookies.sqlite`.
Не переносите и не публикуйте эти файлы: они содержат зашифрованные сессионные
данные множества сайтов.

Cookie имеет срок жизни. Если в логе появится ошибка об отклонённой сессии,
войдите на сайт заново и повторите шаги выше.

## Обновление Python и очистка глобальных пакетов

Для проекта достаточно `start.bat`: зависимости всегда лежат в `.venv`.
Актуальную версию Python для Windows лучше установить с
[страницы Python 3.14.6](https://www.python.org/downloads/release/python-3146/),
после чего удалите и создайте `.venv` заново, запустив `start.bat`.
